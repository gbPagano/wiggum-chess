## Codebase Patterns
- Use `chesslib::analysis::{analyze_fen, StockfishProcess}` for Stockfish evaluation
- `chess-runner` uses clap with `#[derive(Parser)]` for subcommands
- CSV output uses `write_csv_row` helper; each row is appended to the file
- Match runner is in `chesslib::match_runner::{Match, MatchObserver}`
- `SmallRng::seed_from_u64` from the `rand` crate for reproducible shuffles
- `chess-runner` already depends on `rand` crate (added during US-004 or later)
- FEN positions file: one FEN per line, loaded into Vec<String>
