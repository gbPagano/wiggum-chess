## Codebase Patterns
- Use `chesslib::analysis::{analyze_fen, StockfishProcess}` for Stockfish evaluation
- `chess-runner` uses clap with `#[derive(Parser)]` for subcommands
- CSV output uses `write_csv` helper; each row is appended to the file, now includes `start_fen` column
- Match runner is in `chesslib::match_runner::{Match, MatchObserver}`
- `SmallRng::seed_from_u64` from the `rand` crate (`rand = { version = "0.8", features = ["small_rng"] }`) for reproducible shuffles
- `chess-runner` depends on `rand = "0.8"` with `small_rng` feature (added in US-004)
- FEN positions file: one FEN per line, loaded into Vec<String>, shuffle with SmallRng
- `play_games` helper extracted from run_match for reuse with different FEN positions
- When positions_file used: one CSV row per FEN (games from that position), not one aggregate row
