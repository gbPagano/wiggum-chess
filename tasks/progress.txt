## Codebase Patterns
- chess-engine/src/engine.rs contains WiggumEngine struct (renamed from MaterialEngine in US-001)
- cargo not available in shell PATH in this environment; use full path or verify via other means
- All Rust quality checks (cargo test, cargo build) must be done via the cargo binary in PATH if available
- UciEngine mock (shell script in tests) silently ignores unmatched commands — no explicit wildcard needed in case statement; unmatched patterns just continue the while loop
- set_option on UciEngine: concrete method only, NOT on Engine trait; no reply expected per UCI spec
- SPRT LLR uses trinomial model with draw_ratio estimated from actual results; probabilities clamped with EPS=1e-9 to avoid log(0)
- sprt_bounds() and compute_llr() are pub fn so they can be unit tested from within the same file's #[cfg(test)] module
