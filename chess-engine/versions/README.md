# Engine Versions

This directory tracks each released version of the Wiggum Engine, including what changed and how it performed.

## Folder Convention

Each version gets its own subfolder named after the version tag (e.g. `v0.1/`, `v0.2/`).

Each subfolder contains:

- **`CHANGES.md`** — What was implemented or changed in this version: evaluation features, search improvements, bug fixes, etc.
- **`report.md`** — Performance benchmark report generated by `chess-runner version-report`. Includes results vs Stockfish at multiple skill levels and any SPRT comparisons against the previous version.
- **`.gitkeep`** — Placeholder committed until `report.md` is generated.

## Adding a New Version

1. Create `versions/<tag>/` with a `CHANGES.md` describing what changed.
2. Add a `.gitkeep` so the folder is tracked before the report exists.
3. Run `scripts/benchmark-version.sh --version <tag> ...` to generate match results.
4. Run `chess-runner version-report --version <tag> ...` to generate `report.md`.
5. Remove `.gitkeep` once `report.md` is committed.
