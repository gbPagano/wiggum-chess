## Codebase Patterns
- Board.side_to_move() returns Color (Color::White / Color::Black) — use .to_index() for array indexing
- Existing search() in chess-engine/src/search.rs takes (board, depth) and returns (Option<ChessMove>, i32)
- chess-engine main.rs handles UCI loop; go command handling is the entry point for search dispatch
- Add new modules to chess-engine/src/lib.rs with `pub mod <name>` before using them from main.rs
- Run `cargo test --workspace -- --skip gen_files::magics::name` to skip the intentional failing test
