## Codebase Patterns
- Board::default() gives the initial chess position; Board::make_move(m) returns a new Board after applying a move
- Opening book format: plain-text, one UCI move sequence per line, space-separated; `#` comments and empty lines skipped
- `data/` directory is gitignored — use `chess-runner/` or other tracked dirs for repository data files
- git identity must be configured before committing: `git config user.email` and `git config user.name`
- ChessMove::from_uci(s, board) validates a move string against legal moves on a given Board
- Match execution is in `run_match()` / `play_games()` in chess-runner/src/main.rs
