## Codebase Patterns
- Cargo is not pre-installed in devcontainer — install with `curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh -s -- -y`, then `source "$HOME/.cargo/env"`
- Workspace root `Cargo.toml` holds profiles; subcrate `Cargo.toml` files should NOT have `[profile.*]` sections
- The test `gen_files::magics::name` intentionally has `assert!(false)` — it's a dev scratchpad, always fails, not a regression
- Run tests with `--skip gen_files::magics::name` to exclude it; all other 46 tests should pass
- `cargo build --workspace`, `cargo test --workspace`, `cargo bench --bench perft -p chesslib` are the key quality checks
