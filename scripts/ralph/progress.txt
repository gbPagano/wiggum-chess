## Codebase Patterns
- New workspace crates: use `cargo new --lib <name>` from workspace root — it auto-adds to Cargo.toml members.
- Build check: `cargo build --workspace` covers all crates including newly added ones.
- chesslib-simple board model: `squares[rank][file]`, rank 0 = white's back rank. `Square { rank, file }` both 0-based. All types in `src/lib.rs`.
- `Board::apply_move` returns a new Board (immutable pattern). Captures handled implicitly: `replace()` on destination square overwrites any piece there.
