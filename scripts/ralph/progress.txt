## Codebase Patterns
- `chess-runner/src/main.rs` contains all CLI arg structs (`MatchArgs`, `SprtArgs`, `PgnMatchArgs`) and subcommand logic in one file (~1655 lines)
- Default values for CLI args are set via `#[arg(long, default_value = "...")]` clap annotations on struct fields
- `MatchArgs` and `PgnMatchArgs` are separate structs with similar but distinct fields — changes to one don't affect the other
- `write_csv` and `write_sprt_csv` functions are in `main.rs` and write CSV rows with headers on first write
